AnnoCell
Third Year Dissertation Project

AnnoCell was a web-based application built for my third year dissertation project. It was developed using a PHP back-end with a SQLite database with the front-end being the standard HTML, CSS and Javascript we all know and love. The application allowed users to sign up, upload huge multi-gigapixel pathology slide images and then invite other users to digitally annotate the images. Annotations could then be discussed via user created threads or users could contact each other directly through the links that were on display on their application profile.
